Projects and Publications

  1. LABMATE: A Learning Aid Using Augmented Reality

    2022

    Afraim Joseph ; Abin Ooommen Joseph ; Amina Ubaid ; Richy Thampi
    SRISHTI 2022 - 8th National Level Project Exhibition | The Proceedings
    Year: 2022 | Conference Paper | Published by: Saintgits College of Engineering

    This was the Official Project for my final semester of Engineering. An Android Application which helps New Students with Thier Labworks by providing ample information of the Required components. It also gives data on multiple labs and consists of 3D Models of these components which is utilized by Augmented Reality to display it in an informative manner.
    The Base of the application is made using Unity3D and the Augmented Reality components are included using Vuforia Engine. Besides these, Blender was used to edit and create the 3D model and Microsoft Visual Studio was used to add behaviour to the 3d models, C# was the language used to implement this.
